# 6esk v2 Pricing Model: First Principles & Modular Lego Pieces

## 1. The Core Philosophy
6esk is a **Modular Customer OS**. We do not charge per-seat. Instead, we charge for the **Operating System**, the **Lego Pieces** (Modules) stacked on top, and the **Orchestration Work** (Actions) performed by those modules.

---

## 2. Monthly Base Fees (The Stacks)

| Lego Piece | Monthly Fee (ZAR) | Description |
| :--- | :--- | :--- |
| **Core OS (Foundation)** | **R499.00** | Required base. Includes Ticketing, **Email**, Vanilla Webchat, and **Unlimited Human Seats**. |
| **WhatsApp Channel** | **+R499.00** | Enables WhatsApp Business API integration. |
| **Voice Channel** | **+R699.00** | Enables Twilio-powered incoming/outgoing voice calls. |
| **AI Orchestration** | **+R1,499.00** | Enables Dexter AI, Automated ticket handling, and Transcript analysis. |
| **BYO Provider Mode** | **+R499.00** | Unlocks the ability for clients to use their own API keys (OpenAI/Twilio). |

---

## 3. Variable Usage Fees (Orchestration Ops Margin)

Usage fees are per-event and vary by channel to account for differing underlying costs (e.g., Resend vs. Twilio) and compute complexity.

| Action Type | Fee (ZAR) | Internal Cost (COGS) | Our Margin |
| :--- | :--- | :--- | :--- |
| **Email Delivered** | **R0.20** | R0.017 | R0.183 |
| **WhatsApp Delivered** | **R0.99** | R0.85 | R0.14 |
| **Voice Call Handled** | **R2.50** | R1.70 | R0.80 |
| **AI Orchestration** | **R1.00** | ~R0.05 | R0.95 |

*Note: If the client uses **BYO Provider Mode**, they pay the "Internal Cost" directly to the provider (OpenAI/Twilio), but they still pay 6esk the **Margin** as an orchestration fee.*

---

## 4. One-Time Setup Fee (Scope-Based)

The setup fee is **"Contact Sales"** and is calculated internally using the following rubric:

| Scope Factor | Base Impact (ZAR) |
| :--- | :--- |
| **Data Migration** | R1,500 per 10k rows |
| **AI Training/Prompting** | R2,500 per major workflow |
| **Complex Telephony/IVR** | R1,000 per branch/region |
| **Custom API Integrations** | R3,500 per endpoint |
| **Onboarding Workshops** | R1,200 per hour |

---

## 5. Value Proposition to Customer
1. **No Seat Tax:** Add your whole company for free.
2. **Modular:** Only pay for the channels you use.
3. **Transparent:** No hidden markups on AI tokens; just a flat orchestration fee per action.
